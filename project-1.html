  • Connect Four

Tools:

Java, DrJava.

Summary:

A graphical simulation of the board game 'Connect Four' using Java Swing APIs. Player inputs moves via the console. The programmed AI calculates its next move and continues the game until the player or AI wins.

Creation:

Using the compiler/editor DrJava, my partner and I worked for two weeks to graphically simulate a game of Connect Four. The bulk of our project consistent of learning and appropriately utilizing the Java Swing APIs for a consistent visual representation. Implementation of the Connect Four AI had been completed prior.

Challenges:

Learning an entirely new API was a new experience for both of us, as this was one of our first forays into using an externally-written resource (Swing APIs) alongside our own programming skills.

One of the biggest hurdles to overcome, however, was ensuring the 'checker' would be added in the appropriate column upon user input, without overwriting previous movies with faulty logic.
